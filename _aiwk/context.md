# HoneyDo2Done Website - Project Context

## Project Overview

Marketing website for HoneyDo2Done app, focused on converting visitors to app downloads and email signups.

## Key Information

- **Repository**: https://github.com/veebeeyoukay/honeydo2done-web.git
- **Hosting**: Netlify
- **Framework**: Next.js 14 (App Router)
- **Database**: Supabase (same instance as app)
- **Phase**: Phase 1 (MVP Launch)

## Brand

- Colors: Honey Gold (#F5A623), Trust Navy (#1E3A5F), Soft Slate (#64748B)
- Fonts: Inter (body), Outfit (headlines)
- Tone: Warm, empathetic, empowering, witty

## Target Audience

Primary: Overwhelmed capable women, 28-55, South Tampa, $150k+ household income

## Key Features Implemented

- Homepage with all sections (Hero, Emotional Mirror, How It Works, etc.)
- How It Works page
- Pricing page
- FAQ page
- Download page
- Legal pages (Privacy, Terms)
- Email capture API
- Analytics integration (Mixpanel + GA4)
- SEO setup (sitemap, robots.txt, metadata)
